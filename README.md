# pokered-randomizer
Randomize pokémons on Pokémon Red US (md5: 3d45c1ee9abd5738df46d2bdda8b57dc), _may_ work Blue and European/Japanese versions.

# Usage
`$ ./a.out pokemon-red.gb` then enjoy playing it in your favourite gameboy emulator.
